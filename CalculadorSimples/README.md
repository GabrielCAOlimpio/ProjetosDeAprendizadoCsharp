
# Calculadora Simples

## Descrição

Este é um projeto simples desenvolvido em **C#** com o objetivo de revisar e praticar os conceitos fundamentais de **Programação Orientada a Objetos (POO)**.

## Objetivo

Através deste projeto, estou explorando e consolidando conhecimentos sobre:

- **Classes e Objetos**
- **Encapsulamento**
- **Herança**
- **Polimorfismo**
- **Abstração**

## Tecnologia

- Linguagem: **C#**
- Plataforma: **.NET**

## Como Executar

```bash
dotnet run
```
